# scala-cli-maven-surefire-spring-findbugs-cucumber-junit-car-data

## Description
A POC for maven app using junit5
and cucumber framework
 with findbugs
and urefire plugin.

## Tech stack
- scala
- maven
	- spring
  - junit5
  - findbugs
  - surefire
  - cucumber

## Docker stack
- adoptopenjdk/maven-openjdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
